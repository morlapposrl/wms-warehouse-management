## 人工智能WMS系统

先进的仓库管理系统，支持多租户、UDC（装载单元）和智能转移，由 **Morlappo Srl** 开发 - *通过人工智能创新*。

---

## 🚀 主要特性

### ✅ **多租户系统**
- 租户间**完全数据隔离**
- 为每个客户提供专用仪表板
- 产品、移动和库存的独立管理
- 定制化物流计费服务

### 🏗️ **UDC系统（装载单元）**
- **多引用**：每个位置多个UDC
- **多产品**：同一UDC中的不同产品
- **完整条码跟踪**用于优化拣选
- **混乱存储**以最大化效率

### 🔄 **高级转移系统**
- **数量转移**带库存控制
- **完整UDC转移**保持内容
- **可配置原因**带授权工作流
- **高级状态**：已确认、待授权

### 🎨 **现代UI/UX设计**
- **彩色设计系统**：每个部分都有特定渐变
- **高级模态框**：紧凑的2列优化布局
- **原生暗色模式**：完整暗色主题支持
- **流畅动画**：现代过渡和微交互

### 📊 **智能库存**
- **Amazon FBA模型**：物理vs逻辑库存
- **优化的共享存储**节省空间
- **UDC可展开视图**提供完整详情
- **最小库存自动警报**

### 🌍 **完整多语言系统**
- **支持6种语言**：🇮🇹 意大利语、🇬🇧 英语、🇫🇷 法语、🇩🇪 德语、🇪🇸 西班牙语、🇨🇳 中文
- **实时语言切换**：无需重新加载即时切换
- **完整本地化**：本地化日期、数字、货币
- **动态翻译**：管理参数和复数形式

---

## 🛠️ 技术栈

- **前端**：SvelteKit + TypeScript + Tailwind CSS
- **后端**：Node.js with SvelteKit API routes
- **数据库**：SQLite + better-sqlite3（原生SQL查询）
- **验证**：Zod数据安全验证
- **UI/UX**：为平板电脑优化的响应式组件

---

## 🔧 设置和开发

### 先决条件
- Node.js 18+
- npm/yarn

### 安装
```bash
# 克隆仓库
git clone [url]
cd gestionale-magazzino

# 安装依赖
npm install

# 启动开发
npm run dev

# 访问 http://localhost:3002
```

### 生产构建
```bash
npm run build
npm run preview
```
